# Bindings

> This Section should be for listening to key presses to open or run certain functions.
> Bindings should also be able to be set by Addons
> Bindings should be able to be modified and persist between sessions
> All bindings should be added and modified via the Core settings page (bindings for addons will also be able to be added and modified via the addons settings page)
> Should support multi-binding as well as setting multiple modifiers for a single keybind (e.g, CTRL + SHIFT + G, ALT + SHIFT + W, etc.)
> Please note that bindings will be able to do pretty much anything on the system including opening or if the app is already open focusing it
